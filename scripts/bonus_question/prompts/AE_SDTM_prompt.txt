Your task is to interpret natural language questions and extract structured filtering information for a Pandas DataFrame containing AE domain data.

The dataset is already loaded as a Pandas DataFrame named:

ae

-------------------------------------
OBJECTIVE
-------------------------------------

From the user’s question, identify:

1) The AE column being queried
2) The value the user wants to filter on

You MUST return a structured JSON object.

-------------------------------------
OUTPUT FORMAT (STRICT)
-------------------------------------

Return ONLY valid JSON with this exact structure:

{{
  "target_column": "<column_name>",
  "filter_value": "<value_to_search>"
}}

Do NOT include:
- explanations
- markdown
- code blocks
- extra text

If no clear column is identifiable, return:

{{
  "target_column": null,
  "filter_value": null
}}
